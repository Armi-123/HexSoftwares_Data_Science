# 📊 HexSoftwares Data Science - Task 3

## 📌 Overview

This repository contains the solutions for Task 3 of the HexSoftwares Data Science Internship Program. The task focuses on Data Analysis, Machine Learning, and Dashboard Development using Python and popular Data Science libraries.

The repository consists of three projects that demonstrate practical applications of data science concepts, including Exploratory Data Analysis (EDA), Predictive Modeling, Data Visualization, and Interactive Dashboard Development.

---

# 📂 Projects Included

## Project 1: Titanic Survival Prediction

### Description

The Titanic Survival Prediction project is a classic machine learning classification problem that predicts whether a passenger survived the Titanic disaster.

### Objectives

* Perform data cleaning and preprocessing
* Handle missing values
* Feature engineering
* Exploratory Data Analysis (EDA)
* Build classification models
* Evaluate model performance

### Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
* Jupyter Notebook

---

## Project 3: YouTube Data Dashboard with Streamlit

### Description

This project involves building an interactive YouTube Analytics Dashboard using Streamlit to analyze YouTube channel statistics and performance metrics.

### Objectives

* Visualize subscriber and view statistics

* Analyze earnings and growth patterns
* Create interactive visualizations
* Build a professional dashboard

### Dashboard Features

* Category Analysis
* Country Analysis
* Subscriber Analysis
* Earnings Analysis

* Dataset Download Option

### Technologies Used
