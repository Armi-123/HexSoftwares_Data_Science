# 🌸 Data Exploration using Iris Dataset

## 📌 Project Overview

This project focuses on Exploratory Data Analysis (EDA) using the famous Iris Dataset. The objective is to understand the dataset structure, analyze feature distributions, identify relationships between variables, and gain insights through visualizations.

The Iris Dataset is one of the most widely used datasets in machine learning and data science for classification and data exploration tasks.

---

## 🎯 Objective

* Load and explore the Iris dataset
* Understand dataset structure and features
* Check for missing values and handle them if necessary
* Perform statistical analysis
* Visualize feature distributions
* Analyze relationships between variables
* Generate meaningful insights from the data

---

## 📂 Dataset Information

The dataset contains 150 observations of iris flowers belonging to three species:

* Iris-setosa
* Iris-versicolor
* Iris-virginica

### Features

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width
* Species

## 📊 Tasks Performed

### 1. Data Loading

* Imported the Iris dataset
* Loaded data into a Pandas DataFrame

### 2. Data Exploration

* Examined dataset shape
* Reviewed column information
* Generated descriptive statistics

### 3. Missing Value Analysis

* Checked for null values
* Verified data completeness

### 4. Data Visualization

Created various visualizations:

* Histogram
* Box Plot
* Pair Plot
* Correlation Heatmap

### 5. Correlation Analysis

* Examined relationships among numerical features
* Identified strongly correlated variables

---

## 📈 Visualizations

### Histogram

Shows the distribution of numerical features.

### Box Plot

Detects outliers and summarizes feature spread.

### Pair Plot

Visualizes relationships between features and species.

### Correlation Heatmap

Displays correlations between numerical variables.

---

## 🔍 Key Insights

* No missing values were found in the dataset.
* Petal Length and Petal Width show strong positive correlation.
* Iris-setosa is clearly distinguishable from the other species.
* Sepal features show greater overlap among species.
* Petal features provide better separation for classification.

---

## ✅ Conclusion

The Iris Dataset provides an excellent introduction to exploratory data analysis. Through statistical analysis and visualizations, important relationships between flower characteristics and species were identified. The project demonstrates fundamental data exploration techniques commonly used in data science workflows.
