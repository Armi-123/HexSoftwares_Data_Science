# Exploratory Data Analysis (EDA) on Titanic Dataset

## 📌 Objective
To perform data cleaning, analysis, and visualization to extract meaningful insights from the Titanic dataset.

## 📊 Dataset
- Name: Titanic Dataset
- Features: PassengerId, Survived, Pclass, Name, Sex, Age, SibSp, Parch, Ticket, Fare, Embarked

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 🔍 Steps Performed
1. Data Loading
2. Data Cleaning (Handling Missing Values)
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Insight Generation

## 📈 Key Insights
- Female passengers had a higher survival rate than males
- Passenger class significantly affected survival chances
- Most passengers were between 20–40 years old
- Fare and class show correlation with survival

## 📷 Visualizations
- Survival Count Plot
- Gender Distribution
- Age Distribution
- Correlation Heatmap

## ✅ Conclusion
EDA helped uncover key patterns in the dataset, highlighting factors influencing survival such as gender, class, and fare.
